# MapProHammer

**MapProHammer** — редактор карт для библиотеки Map Pro (MadOut games).

Разработан на C# (WPF) пользователем **Catir1337**.

---

## Возможности

- Редактирование 3D-моделей и мешей
- Поддержка форматов Map Pro
- Удобный графический интерфейс (WPF)
